系统设计之如何设计一个新鲜事系统

· 2022-06-24 ·

之前面 tiktok 和 pdd 的时候，问了几个系统设计的问题，之前没有系统性准备过，回答得不太满意。所以回过头来，准备一下系统设计。在某地上看了蛮多系统设计的讨论帖，网上也有很多讲解的课程，为了避免广告嫌疑，不说平台名称了。计划学习一下，记录一点笔记（吐槽一下，课程中某些设计方案以及表设计，槽点蛮多）。

系统设计的 4s 分析法：
Scenario 场景

  • 说人话：需要设计哪些功能，设计得多牛
  • Ask / Features / QPS / DAU / Interfaces

Service 服务

  • 说人话：将大系统拆分为小服务
  • Split / Application / Module

Storage 存储

  • 说人话：数据如何存储与访问
  • Schema / Data / SQL / NoSQL / File System

Scale 升级

  • 说人话：解决缺陷，处理可能遇到的问题
  • Sharding / Optimize / Special Case
